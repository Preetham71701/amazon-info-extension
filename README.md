# Amazon Product Cost Calculator Extension

## Overview

Amazon Product Cost Calculator is a Chrome extension designed for e-commerce businesses importing products from Amazon USA to India.

The extension adds a convenient **SHOW INFO** button on Amazon product pages and instantly calculates estimated landed costs and selling prices based on product weight, dimensions, shipping costs, taxes, and markup percentages.

---

## Features

### Product Information Extraction

* Product Price
* Product Weight
* Product Dimensions

### Logistics Calculations

* Actual Weight Calculation
* Dimensional Weight Calculation
* Formula Weight Selection

### Import Cost Estimation

The extension estimates the total landed cost using:

* Product Cost
* Currency Conversion
* International Shipping
* Domestic Handling Charges
* Platform Charges
* GST Calculations

### Selling Price Recommendations

Automatically generates suggested selling prices for:

* 5% Markup
* 10% Markup
* 15% Markup
* 20% Markup
* 25% Markup
* 30% Markup

---

## How It Works

1. Open any Amazon product page.
2. Click the **SHOW INFO** button.
3. The extension extracts product information.
4. Weight calculations are performed.
5. Estimated landed cost is calculated.
6. Recommended selling prices are displayed.

## Installation

1. Download or clone this repository.
2. Open Chrome.
3. Navigate to:

```
chrome://extensions
```

4. Enable **Developer Mode**.
5. Click **Load Unpacked**.
6. Select the extension folder.
7. Open an Amazon product page and click **SHOW INFO**.
